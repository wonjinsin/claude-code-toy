# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

This is a toy/learning repository used for testing and practice exercises with Claude Code.

## Directory Structure

- **practice/** - Practice exercises and learning materials
- **.claude/** - Claude Code settings and configuration

## Configuration

### Permissions

The repository has Claude Code permissions configured to allow the `keybindings-help` skill for customizing keyboard shortcuts.

### Keybindings

Custom keybindings are configured in `~/.claude/keybindings.json`. See your local keybindings file for current configuration.

## Getting Started

This is a bare repository with no build system, tests, or package dependencies configured. Use it for:
- Learning and experimentation
- Practice exercises in the `practice/` directory
- Testing Claude Code features

## Notes for Future Work
